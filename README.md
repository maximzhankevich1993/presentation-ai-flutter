# Presentation AI Flutter

Flutter Web приложение для создания и редактирования презентаций с помощью Искусственного Интеллекта.

## Возможности

- Создание презентаций по теме (до 10 слайдов бесплатно)
- Автоматический подбор картинок через Unsplash
- Редактирование текста слайдов
- Выбор фона (цвет / градиент)
- Экспорт в PPTX
- Адаптивный дизайн (работает на Web, Windows, macOS, iOS, Android)
